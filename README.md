# yas-grpc
Support grpc call, grpc document and grpc mock server

# Prerequisite
- Installed [`yaml-scene`](https://www.npmjs.com/package/yaml-scene)


# Installation

```sh
  # Install via yarn
  yarn global add yas-grpc

  # Or install via npm
  npm install -g yas-grpc
```
